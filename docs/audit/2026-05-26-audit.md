# Security Audit — 2026-05-26 (pre-0.2.0, M1 hardening pass)

> **Scope**: the M1 playable core (`src/piece.cyr`, `board.cyr`, `world.cyr`,
> `score.cyr`, `framebuf.cyr`, `render.cyr`, `input.cyr`, `tick.cyr`,
> `present.cyr`, `main.cyr`). First P(-1) hardening pass per
> [first-party-standards § Security Hardening](https://github.com/MacCracken/agnosticos/blob/main/docs/development/first-party/first-party-standards.md).
>
> **Result**: no CRITICAL / HIGH / MEDIUM findings. Two LOW (defense-in-depth)
> findings, both fixed in this pass. Bare-stdlib, no external deps, no FFI, no
> shell-out — the attack surface is small by construction ([ADR 0003](../adr/0003-self-rolled-primitives.md)).

## Severity key

CRITICAL (remote / privilege escalation) · HIGH (moderate effort) ·
MEDIUM (specific conditions) · LOW (defense-in-depth).

## Checklist

### 1. Input validation

External inputs in M1: command-line args (`frames`, `seed` via `atoi`) and
stdin bytes (`input_poll`). No file, network, or environment input yet (the
save-file surface arrives in M5).

- `atoi(argv(1))` → `frames`: `run_headless` loops `while (i < frames)`. A
  negative/garbage value yields zero iterations (renders an empty board and
  exits) — safe. A huge value is bounded by the top-out `break`. No overflow
  path. **OK.**
- `atoi(argv(2))` → `seed`: any 64-bit value is a valid RNG seed. **OK.**
- `input_poll` reads ≤ 32 bytes into `buf[32]` and iterates `while (pos < n)`
  with `n` from the `read` return; arrow-sequence look-ahead is guarded by
  `if (pos + 2 < n)`. No over-read. **OK.**

### 2. Buffer safety

Every `var buf[N]` (N = bytes) reviewed:

- `input.cyr`: `orig_termios[64]` / `raw[64]` — `memcpy(...,60)`, `load32/store32`
  at offset 12 (+4 = 16 ≤ 64), `store8` at 21/22 — all in range. `buf[32]`
  read with length 32. **OK.**
- `tick.cyr`: `ts[16]` — two `store64` at +0 and +8 fill exactly 16. **OK.**
- `board.cyr`: `cells = alloc(w*h)` (200 B); every access goes through
  `board_collides` / `board_lock` which bounds-check `cx ∈ [0,w)`, `cy ∈ [0,h)`
  before any `board_set`. Max linear index `(h-1)*w + (w-1) = 199 < 200`. **OK.**
- `framebuf.cyr`: `pixels = alloc(w*h*3)`. `fb_set` is bounds-checked.
  **LOW-1 (fixed)** — `fb_get` was *not* bounds-checked.
- `present.cyr`: `present_buf = alloc(w*h*4)`; blit writes `di ∈ [0, n*4)`. **OK.**

### 3. Syscall review

- `syscall(16, ioctl, ...)` in `input_init` — **LOW-2 (fixed)**: TCGETS return
  was ignored (would proceed with garbage termios on a non-tty).
- `syscall(0, read, &buf, 32)` — return captured, `n <= 0` short-circuits. **OK.**
- `file_open` returns checked in both `fb_write_ppm` (`fd < 0 → -1`) and
  `fb_present_open` (`< 0 → no-op`). **OK.**
- `nanosleep` (35), `lseek` (8), `time` (201) — best-effort, benign return
  values; failure degrades gracefully (a dropped frame / non-varied seed). **OK.**
- `exit` (60) — terminal. **OK.**

### 4. Pointer validation

All heap pointers (`board`, `world`, `framebuf`, `present_buf`) are internally
allocated, never derived from untrusted input. `piece_word` indexes are clamped
by construction (`cur_type ∈ 0..6`, `cur_rot ∈ 0..3`); an out-of-range type
falls through to a defined branch, and `cell_x/cell_y` mask with `& 15` — no
out-of-bounds dereference. **OK.**

### 5. Command injection

No `sys_system` / `exec_*`. **PASS.**

### 6. Path traversal

All file paths are hardcoded constants (`"build/cyrius-polyomino-headless.ppm"`,
`"/dev/fb0"`). No external path input in M1. **PASS.** *(M5 save files will
introduce the first user-influenced path + untrusted file contents; the
sigil-hash-before-load requirement is already specified — re-audit at M5.)*

### 7. Known CVE review

Zero external dependencies — bare Cyrius stdlib only ([ADR 0003](../adr/0003-self-rolled-primitives.md)).
No third-party parsers, no network stack, no codec. Nothing to cross-check
against CVE databases. **PASS.**

### 8. Findings

| ID | Severity | Finding | Status |
|----|----------|---------|--------|
| LOW-1 | LOW | `fb_get` read was not bounds-checked (potential OOB read if ever called out of range; only called with valid coords today). | **Fixed** — added the same `x/y` guards as `fb_set`; OOB returns 0. |
| LOW-2 | LOW | `input_init` ignored the `ioctl(TCGETS)` return; on a non-tty it would set raw mode from uninitialised termios. | **Fixed** — bail with `-1` (leave cooked mode) when TCGETS fails. |
| INFO-1 | — | `frames` arg unvalidated. Negative → no-op; huge → bounded by top-out. | No action; documented. |
| INFO-2 | — | M5 save-file I/O is the first untrusted-file surface (path + contents). | Re-audit at M5; sigil hash-check before load is mandatory. |
